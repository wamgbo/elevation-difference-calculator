📐 Elevation Difference Calculator (高程差計算器)

這是一個專為土木工程測量設計的高程差計算工具。本專案由開發者與 國立中興大學 (NCHU) 土木工程學系 學生共同合作開發，旨在將繁瑣的測量數據計算自動化，減少人工計算誤差。
🏗️ 合作背景 (Collaboration)

本工具核心演算法經由 中興大學土木工程學系 學生進行邏輯驗證與學理指導。結合了土木工程的專業測量知識與現代化軟體開發流程，確保計算結果符合業界規範與學術精確度。
✨ 核心功能

    高程差自動計算：輸入前後視讀數，自動產出高程差。

    誤差修正邏輯：內建土木測量常用的校正演算法。

    簡潔 UI 介面：專為工程現場人員設計，易於操作。
